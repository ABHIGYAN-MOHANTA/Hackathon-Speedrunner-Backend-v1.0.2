Hackathon-Speedrunner API

All these paths are valid only for the allowed origins

  • /users
    GET
    POST(need: username* password* roles active)
    PATCH(need: username* password* roles* active* id*)
    DELETE(need: id*)
  • /auth (need: to be a user, will get automatically authorised by jwt, will get an access token: (15m expiry) and a refresh token: (7d expiry))
    POST(need: username* password*) to login
  • /auth/refresh
    GET(need: cookie => refresh token from jwt)
  • /auth/logout (need: cookie => refresh token from jwt)
    POST(need: cookie => refresh token from jwt)
  • /mlh
    GET(need: auth)
  • /devpost
    GET(need: auth)
  • /posts (need: auth)
    GET (need: auth)
    POST (need: user*, title*, text*, date*, location*, prize*, imagesrc)
    PATCH (need: id*, user*, title*, text*, date*, location*, prize*, imagesrc*, completed*)
    DELETE (need: id*)
